# Learning Analytics Data Science Project

This project applies a complete end-to-end data science workflow to a synthetic learning analytics dataset using Python.

The dataset contains student engagement metrics, academic performance variables, teaching methods, and success outcomes. The project focuses on identifying and cleaning data quality problems before applying exploratory analysis and machine learning techniques.

## Project Tasks

- Data inspection and quality assessment
- Missing value handling and data cleaning
- Categorical normalization and duplicate handling
- Feature engineering and transformation
- Exploratory Data Analysis (EDA)
- Principal Component Analysis (PCA)
- Logistic Regression classification
- Classification performance evaluation
- K-Means clustering

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Main Methods

### Data Cleaning
- Missing value imputation using median
- Duplicate record removal
- Outlier and impossible value handling
- Category standardization
- Regex-based e-mail validation

### Exploratory Data Analysis
- Histogram
- Boxplot
- Bar chart
- Scatter plot

### Machine Learning
- Principal Component Analysis (PCA)
- Logistic Regression
- Confusion Matrix Evaluation
- K-Means Clustering

## Files

- `student-id-HW3.ipynb` → Main Jupyter Notebook
- `student-id-HW3.pdf` → Exported notebook report (optional)

## Project Goal

The goal of this project is to demonstrate a full introductory data science workflow including preprocessing, visualization, dimensionality reduction, supervised learning, evaluation, and unsupervised learning techniques on educational data.
